# ChessMaster ♚

**Built by Abdullah Javid**

A professional, fully-featured chess game — zero install, works in any browser.

## 🎮 Play Live
**[▶ Play Now](https://mabdullahab614-alt.github.io/chessmaster/)**

## ✨ Features
- **Minimax AI** with alpha-beta pruning + piece-square tables (5 difficulty levels)
- **Opening Book** — plays 20+ real chess openings (Ruy Lopez, Sicilian, Queen's Gambit...)
- **Evaluation Bar** — live position advantage indicator
- **Drag & Drop** + 1-click moves + touch support
- **Move Navigation** — step through every move after the game
- **Sound Effects** — move, capture, check, castle, win, lose
- **ELO Tracking** — your rating updates after every game
- **5 Board Themes** — Dark, Green, Blue, Wood, Mono
- **Time Controls** — Bullet, Blitz, Rapid, Classical, Unlimited
- **PGN Export** — copy full game notation
- **Captured Pieces** display with material advantage
- **2-Player mode** — pass and play

## 🚀 Run Locally
Just open `index.html` in your browser — no server, no install needed.

## 🛠 Tech Stack
- Vanilla HTML/CSS/JavaScript
- [chess.js](https://github.com/jhlywa/chess.js) via CDN
- Web Audio API for sounds
- Web Worker for non-blocking AI
